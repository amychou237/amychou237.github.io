    Linux Ubuntu 16.04 下載 Filebeat

    filebeat

    檔案下載

    curl -L -O https://artifacts.elastic.co/downloads/beats/filebeat/filebeat-7.2.0-amd64.deb

    curl(取得網頁內容)

    -O (將網頁特定檔案存至本機)

    -L (可將地址強制重新定向)

    安裝filebeat

    dpkg (Debian Linux用來安裝、創建安裝檔的工具)

    -i (安裝軟體)

    -r (刪除軟體)

    -P (刪除軟體+配置文件)

    設定開機自啟filebeat

    將啟動服務的語法加入/etc/rc.local

    /etc/init.d/filebeat start or sudo service filebeat start
